📌 Sobre o projeto

O We Care foi o meu primeiro projeto de landing page, desenvolvido com o objetivo de aprender os fundamentos do desenvolvimento web.

Neste projeto, foquei em entender a estrutura do HTML, estilização com CSS e primeiros conceitos de responsividade.

Projetos como esse são muito comuns no início da jornada de front-end, servindo como base para evoluir para layouts mais complexos e aplicações completas.

💻 Funcionalidades
- Estrutura básica de uma landing page
- Estilização com CSS
- Layout simples e organizado
- Primeiros conceitos de responsividade

📱 Responsividade

O projeto possui uma adaptação básica para diferentes tamanhos de tela, sendo um dos primeiros contatos com design responsivo.
